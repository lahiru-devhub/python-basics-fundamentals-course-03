## Prerequisites

- Python 3.12+ (recommended)

Check your Python version:

```bash
python --version
```

## How to Run Files

From the project root, run any file with:

```bash
python path/to/file.py
```

Examples:

```bash
python week-01/data_types.py
python week-01/login/login.py
python week-02/loops.py
```

## Learning Flow

1. Start with files in `week-01` to understand basic syntax, variables, and data types.
2. Continue with `week-02` to practice operators, conditions, loops, functions, and error handling.
3. Modify each file and re-run it to observe behavior changes.

## Notes

- Some files are intentionally small and focused on one concept.
- Use comments and print statements while experimenting.
- If a script takes input, run it in a terminal and follow the prompt.
